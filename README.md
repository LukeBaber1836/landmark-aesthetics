# Landmark Aesthetics

This is the official website for Landmark Aesthetics in Jacksonville, TX, built with Next.js.

## Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Content:** Markdown with MDX support

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (version 18.x or higher)

## Key Features

- Server-side rendering with Next.js 15
- TypeScript for type safety
- Tailwind CSS for styling
- MDX support for content management
- Responsive design
- SEO optimized
- Image optimization with Next.js Image
- Animation effects with AOS

## Content Management

Content is managed through Markdown files located in the `src/content/` directory:

- **Blog posts:** `src/content/blog/`
- **Projects:** `src/content/projects/`
- **Pages:** `src/content/pages/`
- **Services:** `src/content/services/`
